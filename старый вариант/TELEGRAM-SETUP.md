# Отправка заявок из Quiz в Telegram (чат тренеров)

В проекте уже есть готовый обработчик для **Cloudflare Worker**:
- `integrations/telegram-cloudflare-worker.js`

И Quiz на сайте умеет отправлять JSON на `SITE_CONFIG.forms.endpoint`.

Ниже — пошаговая настройка, чтобы заявки приходили **в групповой чат тренеров**.

---

## 1) Создай Telegram‑бота и получи токен

1. Открой Telegram и найди бота **@BotFather**
2. Команда: `/newbot`
3. Введи имя бота (любое) и username (должен заканчиваться на `bot`, например `SkillPlusLeadBot`)
4. BotFather выдаст токен вида:

```
1234567890:AAHh..._TOKEN_...xyz
```

Сохрани его — это `TELEGRAM_BOT_TOKEN`.

---

## 2) Добавь бота в чат тренеров

1. Открой **групповой чат тренеров** (или создай новый)
2. Добавь туда бота как участника
3. Важно: в настройках чата убедись, что боту разрешено отправлять сообщения.
   Обычно достаточно просто добавить бота, но если в чате строгие права — сделай бота админом.

---

## 3) Узнай `chat_id` группового чата

### Способ A (самый простой): через getUpdates
1. Напиши в чат любое сообщение (например: `test`)
2. Открой в браузере (с ПК):

```txt
https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates
```

3. В ответе найди кусок:

```json
"chat":{"id":-1001234567890,"title":"Тренеры", ...}
```

Тебе нужен **id** (обычно отрицательный и начинается с `-100...`) — это `TELEGRAM_CHAT_ID`.

> Если `getUpdates` пустой — значит бот не получил апдейты.
> Тогда напиши сообщение в чат ещё раз или временно отключи приватность бота у BotFather:
> `/setprivacy` → выбери бота → `Disable` (не обязательно для отправки, но иногда помогает получить chat_id).

---

## 4) Разверни Cloudflare Worker

### 4.1 Регистрация Cloudflare
1. Перейди на сайт Cloudflare и зарегистрируй аккаунт
2. После входа открой раздел **Workers & Pages**

### 4.2 Создание Worker
1. Нажми **Create application** → **Workers** → **Create Worker**
2. Назови, например: `skillplus-telegram-leads`
3. В редакторе кода вставь содержимое файла:

`integrations/telegram-cloudflare-worker.js`

4. Нажми **Deploy**

### 4.3 Добавь секреты (Secrets)
В Worker открой:
**Settings → Variables → Secrets**

Добавь 2 секрета:

- `TELEGRAM_BOT_TOKEN` = токен от BotFather
- `TELEGRAM_CHAT_ID` = chat_id группы тренеров (например `-1001234567890`)

Сохрани.

---

## 5) Подключи endpoint на сайте

Открой файл:
`assets/js/config.js`

Поставь:

```js
forms: {
  mode: "webhook",
  endpoint: "https://<твое-имя>.workers.dev"
}
```

Где `workers.dev` URL ты увидишь в Cloudflare после деплоя.

---

## 6) Проверка
1. Открой сайт
2. Пройди Quiz до конца
3. Введи имя + телефон
4. Нажми **Отправить заявку тренеру**
5. В чате тренеров должна появиться заявка 🎉

---

## Если что-то не отправляется
- Проверь `mode: "webhook"` и правильность `endpoint`
- Проверь, что секреты в Worker добавлены без пробелов
- Проверь, что бот находится в группе и может писать
- В Cloudflare Worker открой **Logs** и посмотри ошибки
