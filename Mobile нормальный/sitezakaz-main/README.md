# Оптимальные размеры изображений для сайта

## Фоновые изображения
- `heroBg.webp` (десктоп) — 2560x3200 пикселей
- `heroBgmobile.webp` (мобильная версия) — 1440x3200 пикселей
- Масштабирование: `background-size: cover`, `background-position: center top`

## Декоративные элементы
- Мяч (`hero-decor/ball.webp`) — 50-70px (у ног мальчика)
- Бутсы — 80-100px (или убрать, если не получается красиво)
- Мелкие декоративные элементы (звездочки, круги) — 20-50px
- Средние декоративные элементы — 60-120px
- Крупные декоративные элементы — 150-300px

## Контентные изображения
- Фото тренеров (`coaches/`) — 300x400px (портрет)
- Фото выпускников (`graduates/`) — SVG, масштабируется автоматически
- Фото галереи (`gallery/`) — SVG, масштабируется автоматически
- Фото тестирования (`testing/`) — 400x300px

## Иконки в блоках
- `elementsinblock/` — 40-60px (в зависимости от блока)

## Важно
- Не создавать новые файлы изображений
- Все изображения уже оптимизированы и находятся в папке `assets/img/`
- Для контентных изображений использовать атрибут `loading="lazy"`
