## Как вставлять видео (VK / YouTube) в блок «ВИДЕО»

Файл данных: `assets/data/gallery.json`

### VK
1) В VK открой «Поделиться → Код для вставки».
2) В iframe возьми **только** значение `src="..."`.
   Пример:
   `https://vk.com/video_ext.php?oid=-187699303&id=456239098`

3) В `gallery.json` добавь элемент:
```json
{
  "type": "vk",
  "src": "https://vk.com/video_ext.php?oid=...&id=...",
  "caption": "Подпись (необязательно)"
}
```

Важно:
- Не добавляй `autoplay=1`. Даже если добавишь — сайт автоматически нормализует `autoplay` в `0`, чтобы видео не запускалось само при прокрутке.

Дополнительно (важно про конец видео VK):

- VK иногда показывает «предложение/превью другого видео» после завершения ролика.
- Чтобы этого избежать **и при этом не включать бесконечный повтор**, мы используем VK JS API:
  - сайт автоматически добавляет `js_api=1` к VK-embed
  - подгружает `https://vk.com/js/api/videoplayer.js`
  - и останавливает видео **чуть-чуть до конца** (чтобы VK не показал экран рекомендаций)

Где это реализовано:
- `assets/js/vk-embed.js` — хелпер VK.VideoPlayer
- `assets/js/media-cards.js` и `assets/js/media-reviews.js` — подключают хелпер к VK iframe

Если тебе нужен именно **повтор (loop)** — добавь в `gallery.json` у видео:
```json
{ "loop": true }
```
По умолчанию `loop` выключен.

Можно вставлять и так:
- В `src` можно вставить **прямо весь iframe-код** из VK — сайт сам вытащит `src="..."`.

### YouTube
Используй embed-ссылку:
`https://www.youtube.com/embed/VIDEO_ID`

Пример элемента:
```json
{
  "type": "youtube",
  "src": "https://www.youtube.com/embed/VIDEO_ID",
  "caption": "Подпись (необязательно)"
}
```

## Как поменять Яндекс-карту (страница «Локации»)

Файл: `locations.html`

Там есть 2 блока:
- `#ymap-nsk` — Новосибирск
- `#ymap-tlt` — Тольятти

В каждом блоке внутри стоит `<script src="https://api-maps.yandex.ru/services/constructor/1.0/js/?um=constructor%3A...">`.

Подсказка по UX (особенно для mobile):
- параметр `scroll=false` в ссылке конструктора отключает масштабирование/перемещение колесиком/скроллом,
  чтобы карта не “улетала” при случайных жестах.
  При этом кнопки `+ / −` на карте остаются.

Чтобы заменить карту:
1) В Yandex Constructor создай/обнови карту
2) Скопируй новый `script src="..."`
3) Замени `src` в нужном блоке (nsk или tlt)

## MAX (ссылка)

Ссылка MAX настраивается в:
- `assets/js/config.js` → `socials.maxLink` (или `contacts.maxLink`, если добавишь)

Важно:
- MAX больше **не** берётся из `whatsappLink` (чтобы не было “MAX → WhatsApp”).
- Если `maxLink` пустой — MAX показывается, но ссылку нужно заполнить вручную.
